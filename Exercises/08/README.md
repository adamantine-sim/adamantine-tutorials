Post_processor, refinement, and units
=====================================
The goal of this exercise is to set the frequencey and the name of output files,
adaptively refine the mesh, and change the units of the mesh. In order to do
this, you will need to modify the `input.info` file. You can can run the
exercise using `./adamantine -i input.info`. The solution of the exercise is
given in `solution.info`.

Please refer to the documentation in
[post_processor](https://adamantine-sim.github.io/adamantine/doc/input_file.html#post_processor-required), 
[refinement](https://adamantine-sim.github.io/adamantine/doc/input_file.html#refinement-required), and 
[units](https://adamantine-sim.github.io/adamantine/doc/input_file.html#units-optional).
